map() method:-
    * This method creates a new array populated with the results of calling a provided function on every element in the calling array.

    Ex:-
        const data = [
            { id: 1, name: "John" },
            { id: 2, name: "Jane" },
            { id: 3, name: "Doe" }
        ];
        const names = data.map(item => {
            return item.name;
        });
        console.log(names);
        ----------------------------------------
        The output will be:
        ["John", "Jane", "Doe"]
